# 项目路线图 (Roadmap)

本项目开发共分为 4 个主要阶段：

- **第 1 阶段**：下载和构建 Chrono。建立合理的源码目录隔离和自动化脚本，获取一个干净的 Chrono 开发依赖环境。
- **第 2 阶段**：接入 Chrono 到本项目。在自身 CMake 框架中合法引入 Chrono 库，保证 C++ 环境连通。
- **第 3 阶段**：固化 baseline examples。实现基准接触弹跳、碰撞、摩擦等物理验证案例。
- **第 4 阶段**：接入 OpenVDB/NanoVDB 和 SDF-互补接触。以插件/扩展包或者继承重写的方式融入网格与SDF的检测机制，实现完整系统。
