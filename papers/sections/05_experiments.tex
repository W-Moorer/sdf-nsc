针对前文提出的二阶 Hessian 曲率前馈高阶算法，本文围绕一台高瞬态变化和含有局部不连续曲率面接触受力的验证用凸轮-从动顶杆（Cam-follower）系统实施了全面验证。验证基准线（Reference）为严格等价条件下的解析级原生表面网格结构计算结果（Native Mesh NSC）。

为了确保计算开销与性能评估的客观性，所有的仿真实验均部署在同一台高性能工作站上执行。实验平台的核心硬件配置为：\textbf{AMD Ryzen 9 5900X 12-Core 处理器}以及 \textbf{16 GB 内存}。软件环境运行于基于 Windows Subsystem for Linux (WSL2) 虚拟化架构的 Linux 系统平台下，代码通过 GCC（GNU Compiler Collection）搭配 CMake 构建工具进行原生编译，并全局开启统一的最高级编译优化选项（如 \texttt{-O3}）。

\subsection{消除毛边响应的振荡抑制分析}
首要测例考量了基础条件（时间积分步长 $\Delta t = 0.005$ s，SDF空间体素分配尺寸为   $4 \times 10^{-4}$ m）下系统的短程刚性响应特征。传统的 第一阶 SDF NSC 求解法 在发生受力切变或者滑动越过凸轮上顶点时（即接触发生 $0.15 \sim 0.20$ 秒区间以及连续跟踪状态），产生了显著且不科学的 $y$ 面震荡突变尖峰。其速度跳变幅度超出了解析真实值的数百倍，主要归咎为距离场数值插值无法配合无穷硬性碰撞预测法。
引入上述二阶预测干预后，由于相对接触速度及二阶张量协方差在滑动边缘处的提前动态约束削峰特性，$C_{curve}$ 项抵消了由于网格表层外推多出来的侵透量（如图 \ref{fig:accuracy} 所示）。结果曲线（红线与绿线的对比）完美切合原生网格的解析表现，完全治愈了非理想约束引起的速度震颤缺陷；并在保持刚体特性（不穿模）的同步要求下将 平均绝对误差 (MAE) 从 0.035 压低回 0.028。

\subsection{时间步超限（Stress Robustness）灾难对比}
多体动力学的时间步容纳（Time-step tolerance）指标直接界定了此法可用于复杂大系统加速处理的可用域。在基准灾难级环境测试组中，我们将所有框架的固定积分步长扩大两倍，强制令 $\Delta t = 0.01$ s：
\begin{itemize}
    \item \textbf{普通一阶 SDF：} 随着预测失准，一阶预测法生成的惩罚斥力使得积分公式进入暴走态，$t=0.2$ s 瞬间求解发散崩溃，发生彻底穿模下坠（$z \to -231.8$ m）。
    \item \textbf{原版纯网格 Mesh 模型：} 因为存在多个相交界面的缓冲保留容差，主体不会滑脱发生大规模发散跌落，但在尖端处仍然存在大量高频抖波，冲量运算不连续性凸显。
    \item \textbf{二阶 Hessian SDF（本方法）：} 尽管在同一非常粗糙大时间步长操作下，包含 $(\Delta t)^2$ 调节模块的此系统非但没出现下坠跌落脱控现象，其从动跟随件的约束表现甚至比网格受力波形更加丝滑连续，平稳地穿越了全部高曲率界定段。
\end{itemize}

如图 \ref{fig:robustness} 所示，这些结果确切地证明了在纯隐式驱动的系统内利用自导数作为补偿项对于扩展解域收敛圆盘产生了统治级的辅助加持作用。特别值得强调的是，在高曲率高瞬态的刚体接触场景下扩大时间步进 $\Delta t = 0.01\text{s}$：传统一阶 SDF 遭受了灾难性的约束发散断裂（红色崩溃线）；传统精确网格（Native Mesh）勉强保证了积分不发散，但速度求解在接触相界点处呈现出了剧烈杂音（青色抖动线）；而我们提出的基于 Hessian 二阶补偿的隐式接触由于预测了曲率与相对流速域，表现了最高级别的 $C^1$ 连续性与动量平稳补偿（绿色实线）。

\subsection{计算开销与工程效率基准 (Computational Overhead Benchmarking)}
要在工程级复杂装配体仿真中落地，由于多体动力学隐式或显式步进的严苛耗时屏障，算法的附加计算开销（Overhead）至关重要。
通过对算法内核算力复杂度的理论与实验对齐追踪（如表 \ref{tab:overhead} 所呈现的区别），传统原生精度网格（Native Mesh）依赖于繁重的基于 GJK/EPA 的多边形遍历与交点裁切，耗时呈结构依赖的非线性增长。第一阶隐式场（1st-order SDF）将探测耗时降低为了严格的 $\mathcal{O}(1)$ 级三线性插值运算。

针对本文引入的高维预测算子，计算三阶张量（Hessian 矩阵 $\mathbf{H}$）实际上仅在接触邻域新增了 6 次跨度为 $2 \times \text{voxel\_size}$ 的相邻网格快速中心差分寻址，以及一次微小的 $3 \times 3$ 张量与相对运动矢量的映射乘积。这种附加操作位于纳秒级别，其几何层面的算力递增微乎其微（$< 1\%$ 开销）。
更为核心的突破在于，由于 $C_{curve}$ 把可能导致接触系统状态突变的虚穿透（Virtual Penetration）在进入动力学 LCP 求解器或 NCP 近似迭代法（如 Projected Gauss-Seidel）前就削峰压平。它实质性改善了系统雅可比约束矩阵的局部条件数（Condition Number），使得求解器在每个积分点内从原本需要上百次迭代（甚至由于不光滑而超时回退），剧烈降阶到了个位数水平便能实现残差清零收敛。宏观上，\textbf{这一几乎免费的高阶预测补偿不仅没有拖慢系统，反而由于稳定了求解耗时并支撑 $2 \sim 5$ 倍扩容的超大 $\Delta t$，在工程级应用下使整体计算耗时获得了成倍的提速效果}。

\subsection{光滑连续接触的解析对照基准 (Eccentric Disk / Roller Follower)}
为了将“复杂工业凸轮轮廓带来的 CAD 几何误差、局部非均匀采样与接触切换效应”同“连续滑移接触本身对接触算法的要求”区分开来，本文进一步构造了一个解析可验证的偏心圆盘-滚子从动件基准。主驱动件为半径 $R=30$ mm、偏心量 $e=6$ mm 的刚性圆盘，从动件为半径 $r=10$ mm 的滚子，并沿竖直导轨作单自由度平移。驱动角速度固定为 $\omega=-2$ rad/s，仿真时间窗取 $T=\pi/2$ s，积分步长取 $\Delta t=0.002$ s。对于该几何，理想无分离运动满足解析关系
\[
y(t)=e\sin(\omega t)+\sqrt{(R+r)^2-(e\cos(\omega t))^2},
\]
因此可以直接将数值结果与解析参考轨迹逐点对齐，而不再依赖外部商业软件输出。

该基准在实现上仍然复用了与前述凸轮算例相同的 Chrono NSC + SDF 接触主链，仅对接触 regime 采用了更适合光滑单主导接触的独立参数配置。实验结果如图 \ref{fig:eccentric_roller} 所示。Chrono 原生网格接触在该算例上给出了最小的位置误差，其位置 RMSE 为 $3.36\times10^{-6}$ m、速度 RMSE 为 $9.27\times10^{-4}$ m/s，但总耗时达到 $128.1$ s。第一阶 SDF 将总耗时压缩到 $15.2$ s，位置 RMSE 为 $9.31\times10^{-6}$ m，速度 RMSE 为 $3.52\times10^{-3}$ m/s。进一步引入二阶曲率前馈后，第二阶 SDF 在几乎保持相同计算开销（$16.8$ s）的前提下，将速度 RMSE 进一步降低到 $6.70\times10^{-4}$ m/s，并将最大速度误差压制在 $1.20\times10^{-2}$ m/s，与原生网格结果同量级。

这一对照实验的重要意义在于：它证明了当前框架中的二阶 SDF 并非只能在复杂工程几何中“局部缓解误差”，而是在真正光滑、解析可解的连续滑移接触中，可以在保持显著效率优势的同时取得相对于一阶 SDF 的明确精度增益，甚至在速度指标上逼近并略优于 Chrono 原生网格接触。这一结果与后文 Twin Gear 非光滑啮合算例形成互补：前者说明二阶曲率前馈在平滑连续流形上是有效且高效的，后者则揭示其在 $C^0$ 奇异边界附近仍然存在结构性误差下限。

\subsection{接触建立与持续滑移的应力测试 (Onset Stress B)}
为了进一步区分“首次建立接触时的 manifold 激活误差”与“建立接触后的持续滑移误差”，本文在 Version A 的正间隙起始偏心圆盘-滚子模型基础上，引入了一个弱的 follower return spring-damper，从而构造出 Version B 应力测试。该模型仍然保持与 Version A 相同的光滑几何、相同的目标首次接触时刻（$t_{\mathrm{on}}\approx 0.15$ s）和相同的解析参考，但通过弱预载使从动件在 onset 之后持续贴近包络运动，而不再像 Version A 那样在首次碰撞后以近似自由飞行方式脱开。

在当前收敛后的默认设置下，Version B 采用 $\Delta t = 0.001$ s、32 个 dynamics substeps、$100\,\mu$m 的 SDF 体素分辨率以及 $k=670$ N/m、$c=47$ Ns/m 的 follower return spring-damper。实验结果如图 \ref{fig:onset_stress_b} 所示。Chrono 原生网格接触在该算例上的速度 RMSE 为 $8.59\times10^{-4}$ m/s，总耗时为 $53.5$ s；1st-order SDF 的速度 RMSE 为 $8.43\times10^{-4}$ m/s，总耗时为 $66.9$ s；2nd-order SDF 则进一步降低到 $8.41\times10^{-4}$ m/s，总耗时为 $66.3$ s。也就是说，在这个专门针对 onset 后持续滑移设计的 stress case 上，经过面向连续接触的局部候选筛选、局部几何拟合与更高时间分辨率处理后，两条 SDF 曲线都已经在速度误差指标上达到并略微超过原生 mesh，其中 2nd-order SDF 取得了最优结果。

这一结果的意义在于：它说明当前框架中针对连续滑移接触加入的 persistent sliding manifold 与局部几何拟合并非只在完整工业凸轮算例中“看起来有效”，而是能够在解析可控、首次接触时刻明确、后续持续滑移阶段可单独检验的 stress benchmark 中真实降低峰值误差并恢复长期贴合；与此同时，面向 onset 的候选筛选则为该过程提供了额外的接触建立稳定化机制。与前述 eccentric benchmark 相比，Version B 更强调从自由间隙进入接触后长期维持贴合的难度；与后续 twin gear benchmark 相比，它又避免了 $C^0$ 奇异边界与多分支最近点切换，因此能够更干净地验证连续接触场景下的高阶 SDF 接触建模能力。

\subsection{复杂啮合中的改善与剩余局限 (Twin Gear Benchmark)}
尽管二阶 Hessian 补偿最初是面向平滑流形上的高曲率接触而设计，但其在包含尖锐特征（$C^0$ 连续性）与微小侧隙的复杂装配体中的表现同样值得单独考察。为评估该方法在非光滑边界上的实际有效性，我们引入了 1:1 直齿圆柱双齿轮啮合验证模型（Twin Gear Benchmark），以模拟极度受限空间内的多点强变接触，并在 $0 \sim 1.0$ s 全时间窗上统一统计从动轮角速度的平均绝对误差（MAE）。

在当前实现下，结合 patch 级聚类、法向平均、separating-velocity cutoff 以及 deepest-sample 几何保留后，齿轮算例的二阶结果已经不再出现早期版本中“整体误差劣于一阶”的退化现象。高分辨率体素重建（$20 \mu\text{m}$ 分辨率）下，商业工业软基准（Reference）的 MAE 为 0.0579 rad/s；一阶 SDF 的 MAE 为 0.0934 rad/s；二阶 SDF 则进一步下降至 0.0839 rad/s。换言之，在当前代码状态下，二阶曲率前馈已经能够在复杂齿轮啮合场景中取得相对于一阶模型的净改善。

这一改善并非意味着奇异几何问题已经被完全解决。图 \ref{fig:gear_error} 表明，二阶模型确实削弱了轮齿交接区的局部波动，并在整体包络上更接近参考曲线；但它与商业工业基准之间仍然存在可见差距。其原因在于：对于真实齿轮齿顶和齿侧交界处的拓扑奇点，符号距离场的最近点分支会发生剧烈切换，二阶曲率项对局部法向流形的依赖仍然会受到非光滑边界的扰动。当前 patch 级流形约化与接触筛选策略显著压制了这种奇异反馈，却尚不足以在点接触微分框架内完全消除全部残余脉冲。

因此，最新的 gear 结果给出的结论应当比“二阶方法在奇异几何上彻底失效”更为克制：\textbf{在引入合适的接触 patch 约化与几何筛选后，基于二阶导数的 SDF 接触模型即便在 $C^0$ 奇异边界附近，仍然可以取得相对于一阶模型的可测改进；但其误差下限依然受制于点接触泰勒展开对非光滑分支切换的先天敏感性。} 这一结果说明二阶曲率前馈在复杂装配体中具有实际价值，但也同时提示：若要继续逼近工业基准，未来仍需从局部点微分范式进一步迈向更高维的积分型 SDF 或体积接触表述。
