\label{sec:experiments}

本节围绕两个层次展开实验：其一是静态的 wrench-preserving 验证，其二是一步时间推进下的 dynamic-consistency 验证。本文强调，静态合力、合力矩与 CoP 的保持是必要条件，但并不是充分条件；对最终动力学精度起决定作用的，是 dense 接触、reduced support 以及 solver-facing reinjection 在同一时间步内是否产生近似一致的广义冲量与广义速度更新。

\subsection{实验目标}

实验的总体目标是验证：本文方法相较于传统点接触流形维护方式，是否能够在保持 Chrono NSC 全局求解链路不变的前提下，更准确地重建连续接触区域在刚体动力学层面的等效 wrench，并在不同接触类型下保持稳定和一致的高精度表现。

具体而言，实验应回答以下问题：
\begin{enumerate}[label=(\arabic*)]
    \item 在点接触、连续区域接触和长条形接触场景中，本文方法是否能显著降低合力和合力矩误差？
    \item 在接触作用线、CoP 漂移和内部 gap 误差上，本文方法是否优于传统流形维护方法？
    \item 多 patch / 多 subpatch 的自适应 refinement 是否能根据接触类型自动调节模型复杂度？
    \item 在一步时间推进意义下，本文方法是否能同时降低广义冲量误差 $e_I$ 与速度更新误差 $e_V$？
    \item solver-facing reinjection 引入的可行域缺口 $e_R$ 是否已成为主导误差来源？
    \item 在不同时间步长、摩擦系数和几何分辨率下，本文方法是否具有较好的鲁棒性？
\end{enumerate}

因此，本节所有动态结果都应理解为对如下命题的验证：\textbf{瞬时静态等效是必要条件，而一步冲量与速度更新一致性才是最终判据。} 若某一方法在静态误差上已经很小，但在 $e_I$、$e_V$ 或 $e_R$ 上仍然偏大，则应认为它尚未达到本文理论所追求的 solver-consistent contact reduction。

\subsection{实验平台与实现设置}

建议的实验平台如下：
\begin{itemize}[leftmargin=2em]
    \item 动力学后端：Chrono NSC；
    \item 广相与候选对管理：Bullet broadphase；
    \item 几何表征：SDF 或表面导出 SDF；
    \item dense 参考模型：高密度积分点 + 局部微求解；
    \item reduced 模型：本文提出的多 patch / 多 subpatch wrench-preserving reduction；
    \item 对比模型：默认 NSC 点接触流形、固定点数 manifold reduction、无 subpatch 的单 patch 代表点模型。
\end{itemize}

为保证可重复性，应统一以下参数：时间步长、求解器容差、摩擦系数、SDF 分辨率、接触采样密度、误差阈值和最大代表点数等。

\subsection{测试场景设计}

\subsubsection{场景 A：点接触基准测试}

该场景用于验证本文方法在典型点接触问题中不会引入不必要误差。建议采用如下模型：
\begin{itemize}[leftmargin=2em]
    \item 小球--小球对心碰撞；
    \item 小球--平面弹跳；
    \item 低速滚动进入单点碰撞。
\end{itemize}

该类场景下，dense 接触区域本身很小，理论上本文方法应当退化为极少量 patch 与极少量代表点，并与标准点接触结果基本一致。重点评价该方法是否保持原有点接触问题的准确性与数值稳定性。

\subsubsection{场景 B：连续区域接触测试}

该场景用于检验本文方法在面接触或连续区域接触中的 wrench 等效能力。建议采用如下模型：
\begin{itemize}[leftmargin=2em]
    \item 凸轮--从动件接触；
    \item 平板--平板局部压靠；
    \item 曲面块体沿导轨表面滚滑接触。
\end{itemize}

该类场景中，重点观察本文方法是否能正确保持合力、合力矩和接触作用线，尤其是法向 CoP 的稳定性与切向摩擦矩的再现能力。

\subsubsection{场景 C：长条形接触测试}

该场景用于检验多 subpatch 策略的必要性和有效性。建议采用如下模型：
\begin{itemize}[leftmargin=2em]
    \item 齿轮齿面啮合局部接触；
    \item 插销--轴承或轴--孔接触；
    \item 细长圆柱与曲面导轨接触。
\end{itemize}

这类场景的典型特点是接触沿一个主方向强烈延展，且局部法向沿接触线发生变化。对其进行单 patch 约化通常会造成明显力矩误差，因此该类实验特别适合验证本文的多 subpatch 自适应分解策略。

\subsubsection{场景 D：多 patch / 不连通接触测试}

建议构造两个或多个不连通 patch 同时接触的情形，例如：
\begin{itemize}[leftmargin=2em]
    \item 非凸零件与平板的多点支撑；
    \item 带凹槽零件的双边接触；
    \item 多凸起接触足与地面的同时接触。
\end{itemize}

该场景用于验证本文方法是否能在拓扑上正确区分多个 patch，并避免不同接触区域被错误合并。

\subsection{评价指标}

建议采用如下指标：

\paragraph{(1) Wrench 误差}
定义 dense 参考模型给出的 wrench 为 $w_{\mathrm{dense}}$，reduced 模型给出的 wrench 为 $w_{\mathrm{red}}$，则
\begin{equation}
\varepsilon_w = \frac{\|w_{\mathrm{dense}}-w_{\mathrm{red}}\|}{\|w_{\mathrm{dense}}\|+\epsilon}.
\end{equation}

\paragraph{(2) 合力误差与合力矩误差}
可分别定义
\begin{equation}
\varepsilon_F = \frac{\|F_{\mathrm{dense}}-F_{\mathrm{red}}\|}{\|F_{\mathrm{dense}}\|+\epsilon},
\qquad
\varepsilon_M = \frac{\|M_{\mathrm{dense}}-M_{\mathrm{red}}\|}{\|M_{\mathrm{dense}}\|+\epsilon}.
\end{equation}

\paragraph{(3) CoP 误差}
若法向合力非零，则定义
\begin{equation}
\varepsilon_{\mathrm{CoP}} = \|c_{\mathrm{cop}}^{\mathrm{dense}}-c_{\mathrm{cop}}^{\mathrm{red}}\|.
\end{equation}

\paragraph{(4) 内部 gap 误差}
\begin{equation}
\varepsilon_g = \max_{s_i\in\mathcal{S}} \max(0,-g(s_i)).
\end{equation}
该指标用于反映 reduced 点接触是否在 patch 内部遗漏了非穿透约束。

\paragraph{(5) 拓扑保持指标}
对多个 patch 场景，统计 reduced 模型恢复的 patch 个数与 dense 参考模型的 patch 个数是否一致，并记录错误合并与错误分裂次数。

\paragraph{(6) 求解规模与运行时间}
记录最终代表点总数 $N_{\mathrm{red}}$、平均 subpatch 数、每步局部微求解时间、全局求解时间与总仿真时间，用于评估精度与计算代价之间的关系。

\paragraph{(7) 一步广义冲量误差}
记 dense 微求解给出的参考一步冲量为 $\Delta p_{\mathrm{dense}}^\star$，reduced 模型给出的对应一步冲量为 $\Delta p_{\mathrm{red}}^\star$，则定义
\begin{equation}
\varepsilon_I
=
\frac{\|\Delta p_{\mathrm{red}}^\star-\Delta p_{\mathrm{dense}}^\star\|_2}
{\|\Delta p_{\mathrm{dense}}^\star\|_2+\epsilon}.
\end{equation}
该指标直接度量 reduced support 是否保住了 dense subpatch 在当前时间步中的主导冲量效应。

\paragraph{(8) 一步速度更新误差}
设 dense 与 reinjected solver contacts 在同一时间步结束时产生的广义速度分别为 $\nu_{\mathrm{dense}}^{+}$ 与 $\nu_{\mathrm{emit}}^{+}$，则定义
\begin{equation}
\varepsilon_V
=
\frac{\|\nu_{\mathrm{emit}}^{+}-\nu_{\mathrm{dense}}^{+}\|_2}
{\|\nu_{\mathrm{dense}}^{+}-\nu^{-}\|_2+\epsilon}.
\end{equation}
它是本文新的决定性动态指标，用于判断压缩模型是否在目标求解器下保持了一步动力学等效。

\paragraph{(9) Reinjection 可行域缺口}
记 reduced support 对应的一步冲量集为 $\mathcal{I}_{\mathrm{red}}^{h}$，solver-facing reinjection 后离散点接触对应的一步冲量集为 $\mathcal{I}_{\mathrm{emit}}^{h}$，则定义
\begin{equation}
\varepsilon_R
=
\max_{s\in\mathcal{S}_{T}}
\frac{|h_{\mathrm{emit}}^{I}(s)-h_{\mathrm{red}}^{I}(s)|}
{h_{\mathrm{red}}^{I}(s)+\epsilon}.
\end{equation}
该量专门用于识别“reduced support 本身已经足够好，但在重新注入 NSC 点接触后丢失了切向可行域”的情形。

\subsection{对比基线}

建议至少设置以下基线：
\begin{enumerate}[label=(\arabic*)]
    \item \textbf{Baseline-1：}Chrono NSC 默认点接触管线；
    \item \textbf{Baseline-2：}固定点数的接触流形约化，如直接保留少量接触点；
    \item \textbf{Baseline-3：}单 patch 的代表点模型，不进行多 subpatch 细分；
    \item \textbf{Baseline-4：}本文方法的消融版本，例如去掉局部参考 wrench 预测、去掉 sentinel gap 监测或去掉 CoP 误差约束。
\end{enumerate}

通过这些基线，可分别回答“多 patch 是否必要”“多 subpatch 是否必要”“wrench-preserving 是否必要”“局部 dense 微求解是否必要”等关键问题。

\subsection{消融实验设计}

建议进行如下消融：
\begin{enumerate}[label=(\alph*)]
    \item 去掉 $e_{\mathcal{C}}$，仅保留几何矩匹配，观察 wrench 误差变化；
    \item 去掉 CoP 约束或 CoP 误差驱动，观察法向力矩与作用线漂移；
    \item 去掉内部 sentinel 监测，观察 patch 内部是否出现漏穿透；
    \item 将每个 subpatch 的代表点数固定为 3，不允许升阶，观察复杂接触下的误差上升；
    \item 用单 patch 替代多 subpatch，观察长条形接触下的误差变化。
\end{enumerate}

\subsection{闭式球碰撞基准}

\paragraph{结果解读与当前结论}
这三组图给出的结论应当是\textbf{谨慎的正结论加上明确的负结论}，而不是把当前原型直接表述为对全文理论设想的完全验证。

首先，图~\ref{fig:static_dynamic_timeseries} 表明，在接触建立初期以及 support set 发生明显重组的阶段，静态指标与动力学指标确实会同时出现峰值，这说明瞬时的 body-level 静态等效误差并非与动力学误差无关。换言之，若 reduced manifold 在这些关键时刻不能较好保持合力、合力矩和 CoP，则后续的冲量与角速度偏差会被立即放大。就这一点而言，图~\ref{fig:static_dynamic_timeseries} 与本文方法论中的基本预期是一致的。

其次，图~\ref{fig:static_dynamic_correlation} 同时说明，\emph{单一静态指标并不能充分解释全局动力学偏差}。当前原型中，$\varepsilon_F$ 与线冲量误差的整体 Pearson 相关系数约为 $0.003$，$\varepsilon_M$ 与角冲量误差约为 $0.053$，$\varepsilon_{\mathrm{CoP}}$ 与角速度误差约为 $-0.006$，$\varepsilon_g$ 与能量漂移差约为 $0.005$。这些值都很弱，表明把“静态误差小”直接推出为“动力学误差小”是不成立的。更准确的解释是：动力学误差具有路径依赖和累积效应，即使某些后续时间步的瞬时静态误差已回落到很小，系统状态仍可能保留早先误差所造成的偏移。

再次，图~\ref{fig:temporal_coherence_dynamics} 表明，把解释扩展到时序连续性之后，现象仍然只能得到\textbf{部分解释}。当前的 Hausdorff 漂移、平均支撑点漂移和 support churn 与对应动力学指标之间也只呈现有限相关，而没有形成足够强的单调关系。这说明本文当前实现里的时序连续性控制虽然是必要方向，但其定义和实现还不足以完全支撑论文方法部分所追求的“动力学等效”强结论。

因此，若将这些图纳入正文，最合适的结论应当是：\textbf{当前原型已经证明“静态 wrench 等效是必要条件，但不是充分条件”；简单动力学精度依赖于瞬时静态误差与时序连续性的共同作用；而现有实现还不足以把这种关系收敛成强而稳定的统计对应。} 这意味着，这三张图\textbf{符合本文的诊断性预期}，因为它们成功揭示了从静态压缩误差到动力学偏差之间的关键断点；但它们\textbf{尚不符合本文的最终性能预期}，因为它们还没有证明当前实现已经达到论文理论部分所设想的高精度动态等效。

基于上述观察，正文中应把这些结果定位为\textbf{原型阶段的桥接验证与误差归因}，而不是最终有效性的定论。它们一方面支持继续沿“wrench-preserving + temporal coherence”路线推进，另一方面也直接说明了下一步必须补足真正的多 subpatch 自适应细分、局部 dense 微求解、reduced force allocation 以及更强的时序冲量一致性设计。

\subsection{当前原型结果：基线与消融}

在当前代码实现中，我们进一步把基线与消融验证接入了统一的 `variant` 管线，并在同一组静态场景与简单动力学场景上比较以下版本：完整方法（full）、固定 4 点的简化流形（fixed4）、不做 subpatch 的单 patch 版本（single\_patch），以及去掉 dense micro-solver、去掉 $e_{\mathcal{C}}$、去掉 sentinel、去掉 impulse transport、去掉 reinjection-aware acceptance 的多个消融版本。这样做的目的不是追求单一场景上的绝对最优值，而是识别本文理论中的哪些模块已经被实验显式支撑，哪些模块仍然没有解决最后的动态误差。

\paragraph{结果解读}
这些结果首先说明，\textbf{完整方法是当前最稳的整体版本}。在静态场景中，full 通过了全部五个场景的 check，并在曲面 `cylinder\_band` 场景上把最大 $\varepsilon_M$、$\varepsilon_{\mathrm{CoP}}$ 和 $\varepsilon_g$ 控制在 $10^{-5}$ 到 $10^{-4}$ 量级；相对地，fixed4 直接在 `tilted\_square`、`dual\_patch` 和 `cylinder\_band` 上失效，而 single\_patch 虽然在这组静态场景中仍可通过，但其平均压缩比明显更低，说明它依靠更粗糙的表示换取了表面上的静态可行性，并没有保住动力学表现。

其次，\textbf{subpatch 与 dense micro-solver 是当前最明确被实验支撑的两个理论块}。single\_patch 在 `tilted\_plate\_impact` 中把最大角速度误差推高到 $24.228$，明显劣于 full 的 $16.847$；去掉 dense micro-solver 后，同一场景的最大角速度误差进一步上升到 $28.892$。这说明本文提出的“多 subpatch + dense micro-reference”并不是装饰性模块，而是影响接触建立期旋转响应的关键件。

再次，\textbf{$e_{\mathcal{C}}$ 和 impulse transport 也有清晰作用，但作用方式不同}。去掉 $e_{\mathcal{C}}$ 会显著拉坏 impact 场景中的角速度与角冲量误差，表明 support-function / cone 目标确实在帮助约束 reduced support 的局部可行域；去掉 impulse transport 则主要在 slide 场景里恶化轨迹、速度与能量漂移，说明时序冲量传输更偏向改善路径依赖误差，而不是单步静态几何误差。

同时，\textbf{当前这组平板场景对 sentinel 与 reinjection acceptance 的敏感性较弱}。关闭 sentinel 或关闭 reinjection-aware acceptance 后，动态图上的峰值指标几乎与 full 重合；这并不意味着这两个模块无效，而更可能意味着当前两个动态场景还没有充分激发内部漏穿透或 reinjection acceptance 的差异。换言之，这两项仍需要在更复杂的曲面接触、长条形接触和多 patch 真正切换的场景里继续验证。

最后，\textbf{剩余主误差已经收敛到求解器侧的切向线冲量表达问题}。从表~\ref{tab:variant_dynamic_summary} 可以看到，slide 场景的最大线冲量误差在所有 variant 中几乎都停留在 $0.2803$ 附近；这说明目前的顽固误差不再主要来自是否使用 dense micro-solver、是否保留 $e_{\mathcal{C}}$、是否做 impulse transport，而更像是 reduced support 在重新离散成 Chrono NSC 点接触时，对切向线冲量可行域的表达能力仍然不足。也正因为如此，当前基线与消融结果一方面支持本文的大部分方法骨架，另一方面也精确定位了论文最终尚未闭合的那一层。

\subsection{预期结果与讨论重点}

本文预期结果包括：
\begin{enumerate}[label=(\arabic*)]
    \item 在点接触场景中，本文方法不显著劣于默认点接触模型；
    \item 在连续区域接触与长条形接触中，本文方法显著降低合力矩误差与 CoP 漂移；
    \item 多 subpatch 机制对长条形接触和多 patch 接触具有决定性作用；
    \item 在相同全局求解框架下，本文方法能以适中的代表点数显著提升刚体层面的接触等效精度。
\end{enumerate}

讨论时应特别强调：本文方法提升的是 \textbf{body-level wrench 等效精度}，而非逐点压力场精度。因此，实验结果应围绕动力学层面的量展开，而不应把其与完整连续接触压力场重建混为一谈。
